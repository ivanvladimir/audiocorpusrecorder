Requires
========

It requieres:

- python_
- gtk_
- pygtk_ (2.0)
- gst_
- pygst_
- sphinx_ (Documentation only)


Usage
=====

Run
::
    python corpus_rec.py

For help:
::
    python corpus_rec.py -h

Generating documentation:
::
    make html

The documentation is located in _build/html
